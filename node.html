01

Abstract

Node is a gamified, virtual node-running protocol built as a Farcaster Mini App on Base. It lowers the barrier to blockchain infrastructure participation from thousands of dollars in hardware to a few taps on a phone — letting anyone spin up a virtual node, accumulate rewards, and compete in a live leaderboard economy.

Rather than requiring technical expertise or capital-intensive hardware, Node abstracts the complexity of node operation into a social, accessible game layer on top of the Farcaster social graph. Users earn $NODE tokens based on their activity, node uptime, and multiplier status.

Node launched its Epoch 1 as a working proof of concept, reaching 189 registered users and trending on Farcaster Apps. This whitepaper outlines the full protocol vision — covering Epoch 1 learnings, Epoch 2 mechanics, tokenomics, NFT infrastructure, and long-term roadmap.

02

The Problem — Blockchain Participation Barriers

Participating in blockchain infrastructure has historically been the domain of well-capitalized, technically advanced operators. Running a validator or node on most networks requires:

Barrier Current Reality
Hardware cost$500 – $10,000+ for qualifying node hardware
Technical knowledgeCLI fluency, server management, uptime monitoring
Capital stakeMinimum token holdings required (32 ETH for Ethereum)
Ongoing maintenanceSoftware updates, downtime penalties, slashing risk
Geographic accessReliable internet + hardware import feasibility

The result: the average person — especially in emerging markets like Africa — is excluded from infrastructure-level rewards. They can hold tokens or trade, but not participate at the layer that actually secures and runs the network.

Node's thesis: infrastructure participation should be as simple as opening a social app. The rewards should be real. The complexity should be hidden.

03

The Node Solution

Node virtualizes the node-running experience entirely within a Farcaster Mini App. Users register a virtual node, keep it active through daily engagement, and earn $NODE rewards proportional to their activity and node tier.

Core Mechanics

Node's gameplay loop is simple but sticky. Users register their node, maintain uptime through daily check-ins, and are rewarded relative to their tier and engagement level. The social layer — built on Farcaster's open social graph — makes participation visible and competitive.

Why Farcaster?

Farcaster provides three things no other platform offers simultaneously: a crypto-native user base already familiar with wallets and tokens, a Mini App framework (Frames v2) that enables seamless in-app onchain actions, and a decentralized social graph that Node can leverage for referrals, leaderboards, and community growth.

Why Base?

Base is Coinbase's Ethereum L2 — fast, cheap, and deeply integrated with the Farcaster ecosystem. All $NODE token operations, NFT mints, and future staking contracts are deployed on Base, keeping transaction costs near-zero for users.

Epoch 2 opens with a registration phase. Users pay a one-time $1 USDC fee to secure a slot and qualify for the $NODE airdrop at Token Generation Event (TGE). The fee serves as a Sybil-resistance mechanism and funds initial protocol operations. Total slots are hard-capped at 2,000, creating scarcity from day one.

Technical Architecture

Frontend

Node is built as a Farcaster Mini App using Neynar Frame Studio — the native development environment for Frames v2. This enables wallet interactions, cast-gating, and social graph reads directly within the Farcaster client experience without requiring users to leave the app.

Backend & Database

The protocol's backend handles node state tracking, uptime calculations, multiplier logic, and leaderboard aggregation. Node state is persisted in a managed database with daily snapshot updates ensuring reward accuracy.

On-Chain Layer

All financial interactions — airdrop registration ($1 USDC), NFT minting, and future $NODE distributions — are executed on Base. Smart contracts are minimal by design, reducing attack surface and gas costs for end users.

Farcaster Integration

Node reads the Farcaster social graph via Neynar API to power referral tracking, user verification, and social proof features. This integration allows the protocol to Sybil-check participants against Farcaster identity — a meaningful signal of genuine crypto-native users.
